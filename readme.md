# キーワード推薦 API

## 起動方法

`flask --app main run --debugger --reload`

## 使い方(2023/3/2 21:00 時点)

### /

リクエストボディを
`{"sntence": "日記本文"}`
として POST をやると、
｀{
"おっぱい": 0,
"ストレス": 1,
"ハイハイ": 0
}
`
として出現回数を返します。

### /suggest

リクエストボディを
`{"sntence": "日記本文"}`
として POST をやると、
｀{
"おっぱい": 0.6,
"ストレス": 0.3,
"ハイハイ": 0.1
}
`
として日記本文との類似度を、上位 20 単語分返します。
